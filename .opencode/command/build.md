Execute build and lint operations.

Tasks:

1. Run linting: `npm run lint` - identify and fix any issues
2. Build production bundle: `npm run build`
3. Preview production build: `npm run preview`
4. Check for any security issues or code quality problems

Focus on:

- ESLint errors and warnings
- Build warnings or chunk size issues
- SonarJS code quality indicators
- Production bundle size optimization

Provide:

- Lint report with issues to fix
- Build status and any warnings
- Recommendations for code quality improvements
- Deployment readiness assessment
